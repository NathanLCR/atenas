Adding Data

Use Telegram commands to manage data:

  • /add_module name="..." code="..."
  • /add_class title="..." day=mon start=10:00 end=12:00
  • /add_shift start="2026-05-18 16:00" end="2026-05-18 23:00"
  • /add_assignment title="..." due="2026-05-21 23:59"
  • /set_status assignment=ID status=in_progress
  • /set_hours assignment=ID completed=2.5

Or import CSV files:

python3 -m core.academic.importers assignments data/assignments.csv --dry-run
python3 -m core.academic.importers assignments data/assignments.csv --apply
