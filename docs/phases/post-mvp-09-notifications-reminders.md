# Post-MVP Phase 9 — Notifications + Reminders

## Status

Post-MVP optional.

## Goal

Add proactive reminders without making Atenas autonomous in unsafe ways.

## In scope

- Telegram reminders
- deadline alerts
- weekly review prompt
- study block reminders
- missed workload warning
- local scheduler

## Out of scope

- autonomous rescheduling without approval
- external calendar writes
- aggressive notification spam
- cloud push services

## Example reminders

```text
You have NLP CA1 due in 2 days. Planned remaining work: 3h.
```

```text
Your next planned study block starts at 16:30: Deep Learning Report.
```

## Exit criteria

Phase 9 is complete when Atenas can send controlled, useful, non-spammy Telegram reminders.
